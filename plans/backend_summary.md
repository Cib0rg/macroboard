# Резюме архитектуры Backend

## Обзор

Создана полная архитектура backend приложения для управления ESP32-S3 макроклавиатурой, совместимая с прошивкой и обеспечивающая красивый пользовательский интерфейс в стиле Mad Catz.

## Ключевые компоненты

### 1. **MacroKeyboard.Backend** (Background Service)
- Работает как фоновый процесс/служба Windows
- Управляет USB HID коммуникацией с устройством
- Обрабатывает события от устройства (кнопки, энкодер)
- Предоставляет WebSocket API для плагинов (порт 28196)
- Обеспечивает IPC для связи с UI и Tray

### 2. **MacroKeyboard.TrayApp** (Системный трей)
- Иконка в системном трее
- **Двойной клик открывает конфигуратор** ✓
- Быстрое переключение профилей через меню
- Глобальные горячие клавиши (Ctrl+Alt+1-5)
- Уведомления о событиях

### 3. **MacroKeyboard.UI** (Configuration Application)
- **Красивый современный интерфейс** в стиле Mad Catz ✓
- Темная тема с неоновыми акцентами
- Круглые кнопки с эффектом свечения
- Плавные анимации и переходы
- Редактор профилей с визуальным предпросмотром
- Настройка кнопок (действия, изображения, LED)
- Браузер плагинов

### 4. **Plugin System**
- Совместимость с Elgato Stream Deck API
- WebSocket сервер на порту 28196
- Поддержка плагинов на HTML/JS, Node.js, Python, C#
- Адаптация изображений (72×72 → 160×160, квадрат → круг)

## Технологический стек

**Основа:**
- .NET 8.0 (C#)
- WPF для UI (Windows)
- Avalonia UI (опционально для кроссплатформенности)

**Ключевые библиотеки:**
- `HidLibrary` - USB HID коммуникация
- `CommunityToolkit.Mvvm` - MVVM паттерн
- `SixLabors.ImageSharp` - обработка изображений
- `Fleck` - WebSocket сервер
- `H.Pipes` - IPC коммуникация
- `ModernWpfUI` / `MaterialDesignThemes` - современный UI

## Совместимость с прошивкой

✅ Полная поддержка протокола USB HID Raw (64 байта пакеты)
✅ Все команды прошивки реализованы (0x01-0x81)
✅ Обработка всех событий устройства (0xF0-0xFF)
✅ Передача изображений JPEG 160×160
✅ Управление RGB LED и эффектами
✅ Поддержка 10 кнопок и энкодера
✅ Управление профилями (3-5 профилей)

## Архитектура взаимодействия

```
User → TrayApp (двойной клик) → UI (конфигуратор)
                ↓                    ↓
            Backend ← IPC → Backend
                ↓
            USB HID
                ↓
            ESP32-S3 Device
```
