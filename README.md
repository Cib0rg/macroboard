# ESP32-S3 Macro Keyboard Project

Проект макроклавиатуры на базе ESP32-S3 с 10 программируемыми кнопками, каждая из которых имеет круглый дисплей, RGB подсветку и настраиваемое действие.

![Project Status](https://img.shields.io/badge/status-in%20planning-yellow)
![License](https://img.shields.io/badge/license-TBD-blue)

## Обзор

Устройство представляет собой программируемую клавиатуру с визуальной обратной связью, аналогичную Elgato Stream Deck, но с открытым исходным кодом и расширенными возможностями.

### Ключевые особенности

- **10 программируемых кнопок** с индивидуальными дисплеями и RGB подсветкой
- **Круглые дисплеи GC9A01** 160×160 пикселей на каждой кнопке
- **Rotary encoder** для быстрого переключения между профилями
- **3-5 профилей** с сохранением в энергонезависимую память
- **WiFi OTA обновление** прошивки
- **USB HID** - работает как обычная клавиатура + custom HID device
- **Управляющий софт** на C# с красивым интерфейсом

## Технологии

### Прошивка (Firmware)

- **Платформа**: ESP32-S3 N16R8 (16MB Flash, 8MB PSRAM)
- **Framework**: ESP-IDF v5.x
- **RTOS**: FreeRTOS
- **USB**: TinyUSB
- **Язык**: C99

**Готовые компоненты ESP-IDF** (не нужно писать с нуля):
- ✅ Bootloader и OTA - `esp_https_ota()`
- ✅ WiFi - `esp_wifi` API
- ✅ NVS и SPIFFS - встроены
- ✅ USB TinyUSB - интегрирован
- ✅ JPEG декодер - аппаратный в ESP32-S3

**Что нужно написать** (~20% кода):
- ❌ GC9A01 display driver
- ❌ Display multiplexer (4 GPIO)
- ❌ Protocol handler
- ❌ Profile manager
- ❌ Application glue

### Управляющий софт (Software)

- **Платформа**: .NET 8.0
- **UI Framework**: WPF или Avalonia UI
- **Архитектура**: MVVM + Clean Architecture
- **USB Communication**: HidLibrary
- **Image Processing**: ImageSharp
- **Язык**: C#

**Модули**:
- 📱 Main Application - настройка профилей и кнопок
- 🔔 Tray Application - работа в фоне, горячие клавиши
- 🔌 Communication Layer - USB HID протокол
- 💾 Infrastructure - сохранение профилей

## Аппаратные компоненты

### Основные компоненты

| Компонент | Модель | Количество | Примечания |
|-----------|--------|------------|------------|
| Микроконтроллер | ESP32-S3 N16R8 | 1 | 16MB Flash, 8MB PSRAM |
| Дисплей | GC9A01 160×160 | 10 | Круглые, 0.71" |
| Кнопки | Тактовые кнопки | 10 | С колпачками |
| RGB LED | WS2812 | 10 | Адресные |
| Энкодер | Rotary encoder | 1 | С кнопкой |
| Шифраторы | 74HC138 | 2 | Для выбора дисплея |

### Схема подключения

**Дисплеи (SPI)**:
- 4 GPIO для мультиплексора (3 бита адреса + 1 SEL)
- SEL=1 → первый 74HC138 (дисплеи 0-7)
- SEL=0 → второй 74HC138 (дисплеи 8-9)

**Кнопки**:
- 10 GPIO с прерываниями (interrupt-driven, без polling)
- Программный debouncing 10-50 мс

**RGB LED**:
- 1 GPIO через RMT peripheral для WS2812

**Энкодер**:
- 2 GPIO для A/B сигналов
- 1 GPIO для кнопки

## Протокол обмена данными

Устройство общается с ПК через **USB HID Raw** (64-байтовые пакеты).

### Структура пакета

```
[0]    Magic (0xA5)
[1]    Command ID
[2-3]  Payload Length
[4-5]  Sequence Number
[6-61] Payload (56 bytes)
[62]   Checksum (XOR)
[63]   End Byte (0x5A)
```

### Основные команды

- `0x01` PING - проверка связи
- `0x10` SET_PROFILE - переключение профиля
- `0x20-0x22` Передача изображений
- `0x30` SET_BUTTON_ACTION - настройка действия
- `0x40` SET_LED_COLOR - настройка RGB
- `0x60-0x61` OTA обновление
- `0x70-0x71` WiFi настройки

Подробнее: [`plans/protocol.md`](plans/protocol.md)
